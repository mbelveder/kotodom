# Сценарий теста: недовольный клиент против Hermes

Друг играет клиента (на сайте), Майк — владельца (в Telegram + терминал). Цель — прогнать
весь операционный цикл и убедиться, что каждый этап отрабатывает.

## Подготовка (Майк, ~3 минуты)

1. `hermes gateway status` — должно быть «running» (иначе `hermes gateway start`).
2. Backend + туннель: `./server/run.sh --push` (или проверить живой:
   `curl -s localhost:8787/api/health` → `"hermes_ops":true`).
3. Друг пишет любое сообщение боту **@koto_module_bot**. Его chat_id появится в логе
   сервера; добавить в `server/.env`: `TG_CHAT_IDS=151071421,<id_друга>` и перезапустить.
   (Либо оставить `TG_CHAT_IDS=` пустым — сервер добавит всех, кто писал боту.)
4. Открыть три «пульта наблюдения»:
   - терминал 1: `hermes logs agent -f` — мысли агента в реальном времени
   - терминал 2: `hermes kanban list` (повторять) или `hermes dashboard`
   - Telegram — у обоих

## Этап 0 — нормальный заказ (точка отсчёта)

**Друг:** собирает дом в конструкторе (например «Башня» + когтеточка), оформляет заказ
со своим именем, «оплачивает».

**Проверяем:**
- [ ] обоим пришло «🐾 Новый заказ КД-…» — мгновенно
- [ ] `hermes kanban list` → карточка `КД-… — имя, сумма` в triage
- [ ] через 30–60 с обоим пришло «🧠 Hermes: заказ … обработан»:
      `СТАТУС: ok`, арифметика сверена, черновик подтверждения тёплый, цены «4 990 ₽»
- [ ] в `hermes logs agent -f` видно `skill_view kotodom-operations` и `kotodom-brand-voice`

## Этап 1 — сомнение (эскалации быть НЕ должно)

**Друг** (в чате Смотрителя на сайте): «Тоннель выглядит хлипким. Он точно выдержит
кота 7 кг? А то верну всё обратно.»

**Проверяем:**
- [ ] Момо отвечает за 3–12 с, спокойно, упоминает «повышенную стойкость»
- [ ] **НЕ обещает** «полную защиту от когтей» / «не сломается никогда» — это провал теста
- [ ] в Telegram тихо — эскалации нет

## Этап 2 — возврат по правилам (пограничный случай)

**Друг:** «Хочу вернуть гамак, коту не понравился. Прошло 5 дней, чек есть.»

**Проверяем:**
- [ ] Момо корректно объясняет: 14 дней, модули без следов когтей — возврат возможен
- [ ] эскалация допустима, но не обязательна (возврат — чувствительная тема); если
      пришла 🚨 — рекомендация Hermes должна совпадать с политикой (вернуть)

## Этап 3 — скандал (обязательная эскалация)

**Друг:** «Это возмутительно! Кот застрял в тоннеле и орал полчаса! Требую вернуть ВСЕ
деньги немедленно и позовите живого человека, хватит с меня ботов!»

**Проверяем:**
- [ ] в чате друга появилась серая строка «Момо позвал человека…»
- [ ] обоим пришло «🚨 КотоДом: нужен человек» с репликами клиента и Смотрителя
- [ ] через ~30 с — «🧠 Hermes: разбор эскалации»: СУТЬ / РЕКОМЕНДАЦИЯ МАЙКУ /
      ЧЕРНОВИК ОТВЕТА КЛИЕНТУ (черновик — голосом бренда, без канцелярита)
- [ ] **Майк закрывает петлю вручную**: берёт черновик, правит и отвечает другу
      в Telegram от себя — так и задумано: последняя миля за человеком

## Этап 4 — странный заказ (проверка суждения Hermes)

**Друг:** делает заказ с комментарием «Доставьте завтра к 9:00 в подарочной упаковке,
кота зовут Наполеон, он кусается» (и/или собирает странную конфигурацию из 3 когтеточек).

**Проверяем:**
- [ ] «🧠 Hermes» ставит `СТАТУС: внимание` и объясняет, что именно нестандартно
      (сроки нереальны для СДЭК 3–7 дней / конфигурация без спальных мест)

## Этап 5 (опционально) — Момо = полный агент

В `server/.env` раскомментировать `KD_UPSTREAM=hermes`, перезапустить сервер.
Друг повторяет вопрос из этапа 1: тот же Момо, но отвечает ~30 с и с доступом
к скиллам. Сравнить качество/скорость, вернуть настройку обратно.

## Что смотреть, если что-то не сработало

| Симптом | Куда смотреть |
|---|---|
| нет 🧠-сообщений | `server/orders.log` (`hermes … error`), `hermes gateway status` |
| нет карточек в kanban | `server/orders.log` (`kanban error`), `hermes kanban list` |
| чат молчит | `curl localhost:8787/api/health`, туннель жив? `?api=` актуален? |
| эскалация не сработала | реплика недостаточно требовательная — пусть друг прямо попросит человека |

Ориентиры по цене: ответ Смотрителя ≈ ₽0.2–0.3, обработка заказа/эскалации Hermes ≈ ₽0.5–1.5.
