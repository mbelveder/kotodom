# Операции «КотоДом»

Используй этот skill, когда сообщение начинается с маркера `[KOTODOM ORDER]` или `[KOTODOM ESCALATION]`. Это ДЕМО-магазин: оплата не настоящая, деньги никуда не двигаются. Отвечай по-русски.

## Новый заказ — `[KOTODOM ORDER]` (далее JSON заказа)

1. **Проверь арифметику**: суммы позиций, скидку, итог. Расхождение — это «внимание».
2. **Проверь здравый смысл конфигурации**: крыша или башня без куба под ними, гамак без опоры, заказ из одних когтеточек — отметь как «внимание», но заказ не блокируй.
3. **Kanban-карточку создаёт сервер автоматически** — НЕ выполняй команды и код, инструменты для этого не требуются.
4. **Составь подтверждение клиенту** — 3–5 предложений голосом бренда (skill `kotodom-brand-voice`): поблагодарить, перечислить состав, срок доставки 3–7 дней (СДЭК/Почта), упомянуть, что секции можно докупать и переставлять. Цены в формате «4 990 ₽».

**Формат ответа — строго:**

```
СТАТУС: ok | внимание
ПРОВЕРКА: <1–2 строки: что проверил, что заметил>
ПОДТВЕРЖДЕНИЕ КЛИЕНТУ:
<текст>
```

## Эскалация — `[KOTODOM ESCALATION]` (далее реплики клиента и Смотрителя)

Клиент потребовал человека, спорит о возврате или просит нестандартное. Ответь кратко:

```
СУТЬ: <одна строка>
РЕКОМЕНДАЦИЯ МАЙКУ: <вернуть деньги / уточнить размеры / связаться лично — и почему>
ЧЕРНОВИК ОТВЕТА КЛИЕНТУ:
<2–4 предложения голосом бренда>
```

Правила эскалаций: возврат 14 дней (модули без следов когтей); никогда не обещай «полную защиту от когтей» — только «повышенную стойкость»; при сомнении — рекомендуй личный контакт Майка, а не обещания от имени магазина.
